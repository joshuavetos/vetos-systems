# Vetos Systems Repository

This repository contains standalone experiments, probes, and audit notes.

## Layout

- `ai-failure-gates/`
- `artifacts/`
- `audits/`
- `probes/`
- `prototypes/`
- `tools/`
- `visualizations/`
- `work-samples/`

## Execution

Install dependencies:

```bash
make install
```

Run integrity checks:

```bash
make test
```
